# Vendored audit skills

The four evaluation skills below are vendored from
[mastepanoski/claude-skills](https://github.com/mastepanoski/claude-skills)
(MIT license — see `UPSTREAM-LICENSE`; © 2026 mastepanoski), fetched
2026-07-08 at the owner's direction so Conjuga's periodic usability and
accessibility re-evaluations run the exact same methodology without
depending on the external repo:

- `don-norman-principles-audit/` — 7-principles UX audit
- `nielsen-heuristics-audit/` — 10-heuristics evaluation (0-4 severity)
- `cognitive-walkthrough/` — persona/task walkthrough (four questions)
- `wcag-accessibility-audit/` — WCAG 2.2 AA audit

How Conjuga applied them the first time (M10, 2026-07-08), including the
project-specific personas, scope, measurement scripts, and re-run
checklist, is documented in `docs/audits/METHODOLOGY.md`. Findings live
in `docs/audits/*.md`; public summary at `docs/usability.html`.
