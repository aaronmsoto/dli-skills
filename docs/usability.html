🦉 Usability & Accessibility

Queremos que Conjuga sea fácil y justa para todos los niños.

Conjuga has been evaluated in two rounds: Round 1 (2026-07-08) on the original design, and Round 2 (2026-07-10) re-running all four evaluations against the new "Prado" visual redesign. Each finding carries an ID and Found / Fixed dates. This page summarizes scope, findings, and current status across both rounds.

Cognitive walkthrough

Four personas — a DLI 3rd grader on a shared tablet (emerging reader, taps big things first), a DLI K-5 graduate on a phone (fluent app user, skims for the game), a parent checking progress in under a minute, and a teacher printing study sheets and a progress report for folders. Each persona walked through core tasks step by step, rated at every action against the four standard walkthrough questions: (Q1) will the user form the right goal? (Q2) is the correct action findable? (Q3) will they connect the action to their goal? (Q4) after acting, is progress visible?

Five tasks evaluated: (1) first visit → finish a first Elige round, (2) find and play 🎧 Escucha, (3) parent checks progress via the home cards and Informe, (4) teacher prints a study sheet + progress report, and (5) an adversarial task where a learner fidgets with the footer setting toggles mid-round at question 7.

Round 1 result: near-100% first-round success for all personas on all core tasks; the only failure points were CW-1 (unsignified start point) and CW-3 (mid-round toggle restart), both fixed 2026-07-08. Round 2 (Prado): re-running the five tasks, the redesign is a usability gain for the critical first-visit task — the ¡Empieza aquí! ribbon and the pre-selected tense read more clearly in the new look, and CW-1/CW-3 remain resolved. Three new low-severity signifier items surfaced (CW-4 theme/sound behind the unlabeled ☰; CW-5 activity tiles equal visual weight; CW-6 the star-total pill reads tappable) — none impede task completion; logged for owner watch. Success likelihoods equal-to-better than Round 1 across all four personas.
