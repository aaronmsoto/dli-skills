🦉 Conjuga — Documentación / Documentation

Public documentation: how to use the app, how it supports DLI teaching standards, and how it adheres to usability & accessibility best practices.

1 · Cómo usar la aplicación / How to use

Estudiantes: elige un grupo de verbos → un tiempo → y sigue la escalera: 📖 Estudia → 🧱 Práctica → ✅ Elige → ✏️ Escribe → 🧩 Empareja. El reto ⚔️ y 🎧 Escucha te esperan después. Toca 🔍 Pista si necesitas ver la tabla; toca cualquier palabra para escucharla.

Parents: progress lives only in this browser's localStorage — no accounts, no login, no analytics, nothing leaves the device (COPPA/FERPA-friendly by construction). Stars measure the three quizzes + the ⚔️ challenge; 🎧 badges are a separate listening track so devices without a Spanish voice are never penalized; 🧱 Práctica is deliberately unscored. The 🔁 review queue brings skills back after 0-7 days depending on mastery.

Teachers: every study table prints as a classroom sheet (Grupo/Nombre/Fecha header included), and 📄 Informe de progreso prints a one-page star/badge report for folders or conferences. Audio plays pre-recorded Spanish clips (downloaded like images — no text is sent anywhere), falling back to the device's built-in Spanish voice offline. 🎧 Escucha and all audio controls appear whenever either source is available, and hide when neither is. The app works offline after first load on most browsers.

2 · Estándares DLI / Standards support

Pedagogy is grounded in the NBPTS World Languages standards (ECYA-WL), with proficiency levels from the NCSSFL-ACTFL Can-Do Statements: recognition precedes production, feedback is corrective and never punitive, and every activity maps to a mode and Novice level (Interpretive Reading/Listening at Novice Low–Mid for Elige, Empareja, and Escucha; Presentational Writing at Novice Mid–High for Escribe; the preterite/imperfect contrast scaffolds toward Novice High storytelling).

Every screen in the app carries an ℹ️ button explaining, in Spanish-first kid language plus adult English, exactly which standard that page serves. The full per-screen mapping is on the Acerca de / Standards page.

3 · Usabilidad y accesibilidad / Usability & accessibility

In July 2026 the app underwent four formal evaluations — a Don Norman design-principles audit, a Nielsen 10-heuristics evaluation, a persona-based cognitive walkthrough, and a WCAG 2.2 Level AA accessibility audit. Findings, severities, measured contrast ratios, and fix status are summarized on the Usability & Accessibility page.
